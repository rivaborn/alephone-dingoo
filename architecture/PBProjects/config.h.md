# PBProjects/config.h

## File Purpose
Autoconf-generated configuration header defining compile-time feature availability for the AlephOne game engine. Declares which optional libraries (OpenGL, SDL_image, SDL_net) and system headers are available during compilation.

## Core Responsibilities
- Define feature availability flags (HAVE_OPENGL, HAVE_SDL_IMAGE, HAVE_SDL_NET)
- Declare header file presence (SDL headers, POSIX unistd.h)
- Store package metadata (name "AlephOne", version "0.13.0")
- Enable conditional compilation of optional subsystems throughout the codebase

## Key Types / Data Structures
None.

## Global / File-Static State
| Name | Type | Scope | Purpose |
|------|------|-------|---------|
| HAVE_OPENGL | int (1) | global | OpenGL graphics support enabled |
| HAVE_SDL_IMAGE | int (1) | global | SDL_image library available |
| HAVE_SDL_NET | int (1) | global | SDL_net networking library available |
| HAVE_SDL_IMAGE_H | int (1) | global | SDL_image.h header present |
| HAVE_SDL_NET_H | int (1) | global | SDL_net.h header present |
| HAVE_UNISTD_H | int (1) | global | POSIX unistd.h header available |
| X_DISPLAY_MISSING | (undef) | global | X11 display availability (disabled) |
| PACKAGE | string | global | "AlephOne" |
| VERSION | string | global | "0.13.0" |

## Key Functions / Methods
None (preprocessor definitions only).

## Control Flow Notes
Not applicable. This is compile-time configuration. Defines are consumed via `#ifdef` / `#if defined()` conditionals throughout the codebase to include/exclude renderer backends (OpenGL), image loading (SDL_image), and networking (SDL_net).

## External Dependencies
- **Generated by Autoconf** from configure script
- **Dependencies configured**: SDL (core + image + net extensions), OpenGL, POSIX
- **Platform**: Assumes POSIX-like system (unistd.h available); X11 support optional
